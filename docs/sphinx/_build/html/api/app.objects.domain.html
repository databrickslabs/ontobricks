back.objects.domain – Domain management¶

Domain class¶

Session-scoped domain management (non-HTTP).

Use Domain with a DomainSession instance. Routes should construct Domain(session, settings) and call methods.

Return the active ontology, assignment, and related fields for API/query use.

For versioned documents, uses the highest numeric version key (same rule as get_domain_info()). For legacy documents, reads top-level keys.
