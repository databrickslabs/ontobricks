Dual Digital Twin

When you build the Digital Twin, OntoBricks creates two complementary representations:

  • Zero-Copy View — a SQL VIEW in Unity Catalog that maps directly to your source data with no duplication. Used for Quality checks.
  • Graph Database — a LadybugDB embedded graph for Knowledge Graph exploration, GraphQL queries, and MCP access.
