About OntoBricks

Visual R2RML mapping tool for Databricks taxonomy integration

What is OntoBricks?

OntoBricks is a visual tool designed to simplify the creation of R2RML mappings for integrating Databricks datasets with ontologies. It provides an intuitive interface for mapping relational tables to RDF triples, making semantic data integration accessible.

Key Features:
  • Visual mapping interface for Databricks tables
  • Automatic R2RML generation
  • Support for complex relationships and foreign keys
  • Real-time validation and preview
  • Built for Databricks Apps deployment
